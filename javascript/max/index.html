nmdr.
Javascript UI Framework
☰

In Short

nmdr is a collection of lightweight and useful UI components such as table, tree, dialogs, popups, menu etc. based on pure Javascript.
It does not need any library like jQuery or something else. All its components can easily be customized to integrate with any website.
