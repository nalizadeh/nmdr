Welcome to the nmdr lightweight pure Javascript UI framkework.
All components can easily be customized to integrate with any website.


This is the nmdrTabs component

Instructions

To initialize a nmdrTabs use the following code:

var tabs = nmdrTabs('MyTabs');
tabs.addTab(title1, url1);
tabs.addTab(title2, url2);
tabs.build();

Your html should contain a div with id 'MyTabs'
The constructor function takes 1 parameter: the id of the "div" which will be build as tabs
The method "addTab" takes 2 parameters: the title of tab and the url of page, which will be showed within this tab.
